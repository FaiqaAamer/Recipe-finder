# Recipe Finder

A JavaScript-based Recipe Finder web app that uses [TheMealDB API](https://www.themealdb.com/) to search recipes by name, browse by first letter, generate random recipes, and view detailed cooking instructions and ingredients.

---

## Features

- Search recipes by name
- Search recipes by first letter
- Generate a random recipe
- Display recipe images
- View detailed instructions and ingredients

---

## Tech Stack

- **HTML** — page structure
- **CSS** — styling
- **JavaScript** — app logic and API integration
- **TheMealDB API** — recipe data source

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/FaiqaAamer/recipe-finder.git
   ```

2. Navigate into the project folder:
   ```bash
   cd recipe-finder
   ```

---

## Usage

**Option A — Direct open**
Double-click `index.html` to open the app in your default browser.

**Option B — Live Server (recommended)**
Open the project folder in VS Code and run it with the Live Server extension for auto-refresh on changes.

**Interacting with the app**

1. Enter a recipe name in the search bar to find matching meals.
2. Click "Random Recipe" to discover something new.
3. Select a recipe to view its image, ingredients, and cooking steps.
